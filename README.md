This code runs the experiments for measurement-domain sub-image estimation based prediction coding (MSEPC) algorithm. The compressed file named MSEPC.zip contains the main function, WaveletSoftware, the useful functions and the test images. For more details of this scheme, please refer to the following paper. Bo Zhang, Zikai Song, Di Xiao, Junqing Yu, and Wei Yang*, "Enhanced Measurement Coding System with Sub-image Estimation for Block Compressive Sensing Images," IEEE Transactions on Multimedia, 2026, accepted. Originally written by Bo Zhang(email: zhangbohust@hust.edu.cn), Huazhong University of Science and Technology.
